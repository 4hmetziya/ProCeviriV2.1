========================================
SUNUCUYU BAŞLATMA TALİMATLARI
========================================

1. YÖNTEM 1 - Batch Dosyası ile (Kolay):
   - START_SERVER.bat dosyasına çift tıklayın
   - Otomatik olarak doğru klasörde başlatır

2. YÖNTEM 2 - Manuel (Terminal/PowerShell):
   - Terminal/PowerShell'i açın
   - Şu komutu çalıştırın:
   
     cd c:\PDF_TRANSLATOR\ProCeviri_V2
     python app.py

3. YÖNTEM 3 - VS Code/Cursor Terminal:
   - Terminal'i açın (Ctrl + `)
   - Şu komutu çalıştırın:
   
     python app.py

ÖNEMLİ:
- Sunucu mutlaka "c:\PDF_TRANSLATOR\ProCeviri_V2" klasöründe çalıştırılmalı
- Bu klasörde app.py ve fonts/ klasörü olmalı
- Fontlar fonts/ klasöründe olmalı (tr_font_reg.ttf, tr_font_bold.ttf, vb.)

Sunucu başladıktan sonra tarayıcıda şu adresi açın:
http://localhost:5000
